# when-to-sort-lines

- **Slug**: when-to-sort-lines
- **Headline**: Sort lines before you clean text and mistakes get easier to see
- **Emotion**: Order
- **Promise**: Make sorting feel like the step that reveals hidden cleanup problems.

## Prompt

Create a 16:9 discovery image for a guide about sorting lines of text. Show scattered paper strips sliding into neat ordered rows on a desk while an adult reviewer spots one outlier strip. The emotion is order and relief: messy pasted text becomes reviewable. Clean modern editorial lighting, no readable text, no app UI, no logos.

## Negative prompt

No readable words, no spreadsheet screenshot, no code editor, no watermark, no robotic hands, no excessive infographic arrows.
